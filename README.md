## Calculator Function

This `calc()` function supports the following operations:
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Floor Division (`//`)
- Modulus (`%`)
- Exponentiation (`**`)

### Usage
```python
print(calc(10, 3, '+'))  # Output: 13
print(calc(10, 0, '/'))  # Output: Error: Division by zero is not allowed.
